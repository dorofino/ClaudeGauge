## Name
Claude Session Key Helper

## Short Description (132 chars max)
One-click setup for the ClaudeGauge device. Auto-fills your Claude.ai session key — no developer tools needed.

## Detailed Description
Claude Session Key Helper makes setting up your ClaudeGauge device effortless.

Instead of opening developer tools and hunting through browser cookies, this extension reads your Claude.ai session key with one click and auto-fills it on your device's configuration page.

HOW IT WORKS:

1. Log into claude.ai in your browser
2. Open your ClaudeGauge device's config page (http://your-device-ip)
3. Click the "Auto-fill from Claude.ai" button that appears on the page
4. Your session key is filled in automatically — just click Save!

You can also click the extension icon in the toolbar to copy your session key to the clipboard.

FEATURES:
- One-click auto-fill on the device config page
- Popup with copy-to-clipboard for manual use
- Only reads the sessionKey cookie from claude.ai — nothing else
- No data is sent anywhere except to your local device
- Works entirely offline (no external servers)

PRIVACY:
This extension only accesses the sessionKey cookie from claude.ai. It does not collect, store, or transmit any data. The session key is only used to fill in the form field on your local device's configuration page.

REQUIREMENTS:
- An active Claude.ai account (logged in)
- A ClaudeGauge device on your local network
